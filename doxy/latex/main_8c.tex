This program implements an application that controls the light intensity of the region. 

@main \mbox{\hyperlink{main_8c}{main.\+c}}

It has 2 modes\+:
\begin{DoxyItemize}
\item Manual mode\+: The user can decide the light intensity with the buttons
\item Automatic mode\+: The user choses the desire light intensity and the program will make the intesity always the same.
\end{DoxyItemize}

On manual mode\+:
\begin{DoxyItemize}
\item But1\+: Manual mode chosen.
\item But2\+: Autumatic mode chosen.
\item But3\+: Decrease led light intensity
\item But4\+: Increase led light intensity
\end{DoxyItemize}

On automatic mode\+:
\begin{DoxyItemize}
\item But1\+: Manual mode chosen.
\item But2\+: Autumatic mode chosen.
\item But3\+: Decrease region light intensity
\item But4\+: Increase region light intensity
\end{DoxyItemize}
